# Observable Backend Platform

A production-style backend observability and monitoring project using:

- Node.js
- Express.js
- Prometheus
- Grafana
- PM2

---

# 🚀 Project Goal

The goal of this project is to implement backend observability by collecting application metrics and visualizing them using Prometheus and Grafana.

This project demonstrates real-world monitoring architecture used in modern DevOps environments.

---

# 🚀 Architecture

Client Requests
↓
Node.js Backend
↓
Prometheus Metrics Collection
↓
Prometheus Server
↓
Grafana Dashboards

---

# 🚀 Features Implemented

## Application Instrumentation
- Prometheus metrics integration
- Default Node.js metrics collection
- HTTP request monitoring
- API response tracking

## Monitoring Dashboards
- Total API Requests
- API Response Time
- API Request Rate
- API Error Monitoring
- Backend Memory Usage

## Process Management
- PM2 process monitoring
- Backend uptime management
